本项目提供了一套专为**安徽大学人工智能学院嵌入式系统课程**量身定制的实验报告生成提示词（Prompt）模板。

## 🛠️ 环境准备与安装

Claude Code 是一个直接运行在终端中的 CLI 工具，能够精准理解项目结构并自动化操作文件。

### 1. 基础依赖环境
* **Node.js 与 npm：** 请确保系统已安装最新版 Node.js。在终端运行 `node -v` 和 `npm -v` 验证。
* **Python 依赖：** 报告文档生成需依赖 `python-docx` 库，在终端执行以下命令安装：
  ```bash
  pip install python-docx

```

### 2. 全局安装 Claude Code

打开系统终端（CMD、PowerShell 或 macOS/Linux Terminal），运行：

```bash
npm install -g @anthropic-ai/claude-code

```

### 3. 初始化与账户授权

执行以下命令启动 CLI 工具：

```bash
claude

```

首次运行时，系统将自动弹出浏览器窗口提示登录 Anthropic 账户并完成 OAuth 授权。完成授权后即可返回终端畅快使用。

## 🚀 快速上手：生成实验报告

本仓库的核心工具是一套深度优化的提示词文件（如 `v1.5.md`），只需将其与项目源码同时提供给 Claude Code 即可。

### 第一步：准备问卷信息

打开本仓库中最新的提示词文件（如 `v1.5.md`），翻至文档末尾的 **“11、报告前必填信息问卷”**。
得益于 v1.5 的自动化解析能力，你只需提供必要的路径与基本参数（如 Q1、Q5、Q6），其余的实验目的、实验环境与底层逻辑均由 AI 从源码中深度提取。

### 第二步：启动 Claude Code

进入代码项目根目录或存放实验模板的目录，打开终端并运行：

```bash
claude

```

### 第三步：发送生成指令

在交互界面中输入提示词指令（请根据个人实际文件路径修改内容）：

```text
请读取你当前目录下的 v1.5.md 提示词模板。根据模板要求，我现在向你提供本次实验的问卷答案：

Q1：实验次序：第四次实验；项目名称：定时器实验；日期：2026.6.30；地点：竹园1129。
Q5：硬件原理图路径：'D:\Desktop\原理图.png'
Q6：主程序路径：'D:\Desktop\main.c'
Q8、Q9、Q10、Q11 均选择'是'（全自动生成）。

请使用 python-docx 库严格按照模板规范，为我生成这份实验报告并保存为 docx 文件。

```

### 第四步：等待文档生成

Claude Code 将自动完成以下全链路操作：

1. 读取并解析 `main.c` 等源文件逻辑。
2. 自动构建逻辑链路并渲染生成 Mermaid 架构图。
3. 调用 Python 脚本精确排版并输出 `.docx` 格式的完整实验报告。
